The built-in DOM methods are great, but they’re not very easy to use. Because of this, many developers use a set of tools called jQuery to access and manipulate the DOM tree. jQuery is a JavaScript library—a collection of related tools (mostly func- tions) that gives us, in this case, a simpler way to work with DOM elements.

To load it into the page, just add <script src="https://code.jquery.com/jquery-2.1.0.js"></script>
